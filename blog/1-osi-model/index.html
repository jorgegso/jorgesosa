2022-01-29

Basic networking for pentesting...

        El modelo de interconexión de sistemas abiertos, es un sistema jerárquico de 7 capas que muestra el funcionamiento de la red, ayuda el entendimiento de su funcionamiento y poder identificar su parte de una forma fácil.

 

  • 7 data == application layer |
  • 6 data == presentation |> the web browser.
  • 5 data == session |
  • 4 segments == Transport layer: donde esta tcp/udp,
  • 3 packets == network
  • 2 frames == datalink
  • 1 bits == Physical layer: si no ESTA no nectada no anda.
